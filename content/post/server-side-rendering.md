Apa itu SSR
teknik menampilakn informasi pada website yang prosesnya di lakukan pada sisi server

kelbihan dari SSR
+ social media sharing dengan adanya thumnail
+ bagu untuk SEO
+ mengurangi dependensi terhadap javascript

kekurangan 
- membutuhkan server yang komplex
- selalu memuat ulang ketika transisi halaman
- latensi lebih besar
